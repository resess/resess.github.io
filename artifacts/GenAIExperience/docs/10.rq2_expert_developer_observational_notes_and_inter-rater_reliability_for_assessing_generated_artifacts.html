Inter-rater Reliability for Assessing Generated Artifacts

The two experts started by inspecting the code for both versions of each scenario. To keep the workload manageable, they selected for further analysis one of the versions, in which the code appeared to be more correct and complete. Then, the experts independently assessed the quality of the generated code of the selected version, for each scenario. They met to discuss their assessment results and resolved disagreements, when possible (approximately 5% disagreement rate). Disagreements that could not be resolved were discussed and resolved with an arbitrator, who is also an author of the paper.
