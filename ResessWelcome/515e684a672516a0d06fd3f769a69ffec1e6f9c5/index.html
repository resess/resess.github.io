    • Two options:
      • GNU Screen
      • Tmux
    • GNU Screen
      • Start screen: screen
      • Detach screen: CTRL-A-D
      • Re-attach latest screen: screen -r
      • Re-attach screen by ID: screen -r [ID]
      • List screen sessions: screen -ls
    • Tmux
      • Start tmux session: tmux new -s [TAG_NAME]
      • Access existing tmux by ID: tmux a [ID]
      • Access existing tmux by tag name: tmux a -t [TAG_NAME]
      • List tmux sessions: tmux ls
      • Kill tmux session: tmux kill-session -t [TAG_NAME]

    Paper Writing

    Organize contents of your research paper into the followings latex files(.tex):

    • paper.text
      • This file should contain the title of your paper, the headers, and inclusion of the files that you need for your paper (as below).
    • macro.tex
      • This file should contain all packages, commands, etc. (just copy from the template)
    • abstract.tex
    • intro.tex
    • background.tex
    • approach.tex
    • results.tex
    • related.tex
    • conclusion.tex
    • 00.bib
      • This file should contain the references to research papers, journals, etc. Bibtex entries of the references should be in this format: last:names:of:all:authors:venue:year.
      • For exmaple, this paper should be referenced as:
        • @InProceedings{Hao:Liu:Nath:Halfond:Govindan:MobiSys:2013,
        ...
        }
    • Make file
      • A make file to build the latex package (just copy from the template). Please refer to Research paper template. Run "make" before you commit and make sure you paper compiles every time you commit.

    Note- Please start each sentence in new line in latex files for better reviewability
